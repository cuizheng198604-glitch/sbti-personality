# SBTI × Big Five 混合卡牌抽卡系统设计规格

> 参照 mbti 抽卡模式，以 SBTI 四维为骨架、大五人格为数值引擎、
> 合并稀有度与角色立绘体系，构成一套完整的卡牌生成规则。

---

## 一、核心理念

| 层级 | 来源 | 作用 |
|------|------|------|
| **SBTI 四维** | SBTI (Split Big Five Trait Inventory) | 卡牌类型名 + 角色立绘方向 + 天赋技能 |
| **大五人格** | Big Five / OCEAN | 数值属性（攻防生命等）+ 隐藏潜力值 |
| **稀有度** | mbti 抽卡模式 | 决定立绘稀有度、附加属性条数、是否有专属技能 |

## 二、SBTI 四维详细定义

```
┌─────────────────────────────────────────────────────┐
│                   SBTI 四维结构                       │
├─────────────┬───────────────────────────────────────┤
│  Energy     │ E (外向活跃) —— I (内向沉稳)            │
│  Awareness  │ S (务实细节) —— N (直觉抽象)            │
│  Decision   │ T (理性逻辑) —— F (情感共情)            │
│  Execution  │ J (计划执行) —— P (灵活探索)            │
└─────────────┴───────────────────────────────────────┘
```

### 2.1 各维度的 Big Five 映射

| SBTI 维度 | 对应大五子维度 | 含义 |
|-----------|--------------|------|
| Energy = E | Extraversion (活跃性) | 合群、活力、主导倾向 |
| Energy = I | Introversion (内省性) | 独立、沉思、低社交刺激需求 |
| Awareness = S | Conscientiousness (尽责性) 低分 | 关注具体现实、经验主义 |
| Awareness = N | Openness (开放性) 高分 | 偏好抽象思维、想象力丰富 |
| Decision = T | Agreeableness (宜人性) 低分 | 逻辑优先、不易受情感影响 |
| Decision = F | Agreeableness (宜人性) 高分 | 共情优先、重视人际和谐 |
| Execution = J | Conscientiousness (尽责性) 高分 | 计划性、自律、偏好确定性 |
| Execution = P | Openness (开放性) 高分 + 低尽责性 | 适应性、好奇心、偏好灵活性 |

### 2.2 基础类型组合（16 + 扩展）

SBTI 本身理论上产生 **16 种基础类型**（同 MBTI），
但因 Awareness 和 Execution 均与 Big Five 有双重映射，可能出现 **中间型/混合型**：

| 分类 | 类型前缀 | 说明 |
|------|---------|------|
| 典型型 | ENTP、ISFJ 等 | 四维全部偏向某一端 |
| 混合型 | ENxP、IxTJ 等 | 其中一维处于中性带（40-60% 倾向） |
| 极端稀有型 | xNTx、xNFx | 同时携带 N + T/F 高分，概率极低 |

## 三、Big Five 数值属性体系

### 3.1 六维战斗属性

| 属性名 | 计算来源 | 说明 |
|--------|---------|------|
| **攻击力** | Openness × 2 + Conscientiousness | 开放性（创意）+ 尽责性（精确） |
| **防御力** | Agreeableness × 2 + Conscientiousness | 宜人（亲和）+ 尽责（稳定） |
| **生命力** | Neuroticism 反算 + Extraversion | 情绪稳定性 + 活力 |
| **速度** | Extraversion + Openness | 外向（行动快）+ 开放（思维快） |
| **暴击率** | Openness + Neuroticism | 高开放（创意）+ 高神经质（不稳定爆发） |
| **闪避率** | Introversion + Openness | 内向（隐蔽）+ 开放（灵活思路） |

### 3.2 大五各维度标准化

```
原始分数（0-5 或 0-100）→ 标准化为 0-100 分
再映射到战斗属性公式：
  属性值 = round(维度分 × 6 + 基础值)
  基础值 = 300（保证所有卡牌属性在 300-900 区间）
```

### 3.3 大五测试题目分组（每维度5题，共25题）

| 维度 | 关键词 | 高分特征 |
|------|-------|---------|
| Openness | 想象、创意、好奇、审美 | 想象力天赋 +2% 暴击率 |
| Conscientiousness | 自律、计划、可靠 | 执行力天赋 +2% 防御力 |
| Extraversion | 社交、活力、主导 | 领袖气质 +2% 攻击力 |
| Agreeableness | 信任、利他、合作 | 亲和力天赋 +2% 生命值 |
| Neuroticism | 焦虑、情绪波动、敏感 | 情绪爆发天赋 +3% 暴击率 |

## 四、稀有度体系（沿用 mbti 抽卡框架）

### 4.1 稀有度定义

| 稀有度 | 概率 | 典型类型 | 立绘品质 | 附加属性条 |
|--------|------|---------|---------|-----------|
| **Common** | 50% | ESFJ、ISFJ、ESTJ、ISTP | N卡 | 0 条 |
| **Rare** | 25% | ENFP、INTP、ENTJ、ISFP | R卡 | 1 条 |
| **Special** | 15% | INFP、INFJ、ENFJ、INTJ | SR卡 | 2 条 |
| **Legendary** | 10% | 全N型（xNxJ/xNxP）+ 全F型（xFxJ）| SSR卡 | 3 条 + 专属技能 |

> **判定逻辑：**
> 1. 抽到 Legendary 时，若同时满足 N（直觉）+ 高开放性 + J/P 极端偏向 → 额外+1条专属属性
> 2. 抽到 Special 时，若大五总分 > 80 → 升级为 SR+

### 4.2 稀有度立绘风格

| 稀有度 | 主色调 | 边框风格 | 特效 |
|--------|-------|---------|------|
| Common | 银灰 | 直线单层 | 无 |
| Rare | 蓝金 | 直线双层 + 微光 | 淡淡光晕 |
| Special | 紫金 | 斜杠花纹 + 内发光 | 流光粒子 |
| Legendary | 虹彩 | 多层立体浮雕 + 动态光效 | 全屏星尘 + 专属背景 |

### 4.3 大五总分稀有度加成

| 大五总分 (0-500) | 稀有度提升 |
|-----------------|-----------|
| 400-500 | Rare → Special，Special → Legendary |
| 300-399 | 保持当前稀有度 |
| 150-299 | Special → Rare |
| 0-149 | 保持当前稀有度 |

## 七、抽卡模拟器实现概要

### 7.1 输入
- SBTI 四维测评结果（每维度 0-100 的连续分数）
- 大五人格测评结果（每维度 0-100 的连续分数）

### 7.2 抽卡算法

```
1. 随机抽取稀有度概率（Common 50 / Rare 25 / Special 15 / Legendary 10）
2. 根据稀有度确定可抽类型池
3. 结合 SBTI 维度和 Big Five 分数生成具体类型
4. 计算六维战斗属性
5. 匹配立绘风格 + 添加特效
6. 附加技能（根据稀有度和 Big Five 高分解锁）
7. 输出完整卡牌数据
```

### 7.3 星级评定

| 稀有度 | 星级 | 综合分计算公式 |
|--------|------|--------------|
| Common | ★★☆☆☆ | 大五总分 × 0.5 + 稀有度基础分 |
| Rare | ★★★☆☆ | 大五总分 × 0.7 + 稀有度基础分 |
| Special | ★★★★☆ | 大五总分 × 0.85 + 稀有度基础分 |
| Legendary | ★★★★★ | 大五总分 × 1.0 + 稀有度基础分 + 技能加成 |

## 九、后续扩展方向

1. **合卡系统**：两张同类SBTI卡合成，下一代继承强化属性
2. **天赋树**：基于大五子维度展开，每个维度3层天赋
3. **羁绊系统**：特定类型组合激活额外属性（例：INTJ+ENFP 激活"思维共振"+15%暴击）
4. **称号系统**：累计特定稀有度抽取次数，解锁成就称号并获得永久属性加成
