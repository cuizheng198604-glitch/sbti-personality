### 核心设计逻辑

**稀有度评级规则：**
- Legendary：大五总分≥400 或 SBTI极端维度≥3个(≥40分差) 或 N直觉高分组合
- Special：大五总分≥300 或 双极端维度
- Rare：大五总分≥220 或 有1个极端维度
- Common：其余

**SBTI → MBTI 映射：**
Energy>50→E / Awareness>50→N / Decision>50→F / Execution>50→P

**Big Five → 战斗属性公式：**
- 攻击 = 320 + (Openness×2 + Conscientiousness)/3×200
- 防御 = 320 + (Agreeableness×2 + Conscientiousness)/3×200
- 生命 = 320 + (情绪稳定分 + Extraversion×0.5)×300
- 速度 = 320 + (Extraversion + Openness×0.8)×200
- 暴击率 = (Openness + 情绪稳定)×12%
- 闪避率 = (1-Extraversion + Openness)×10%

**天赋技能解锁：**
- SBTI 最极端维度 → 1个维度技能
- Big Five≥70分 → 触发对应技能（≥80为高级版）

**角色立绘方向（16种）：**
INTJ建筑师/INTP策划者/INFJ预言家/INFP理想者/
ISFJ守护者/ISFP艺术家/ISTP匠人/ISTJ审计官/
ENTJ指挥官/ENTP发明家/ENFJ导师/ENFP冒险家/
ESFJ执政官/ESFP表演者/ESTJ管理者/ESTP企业家
